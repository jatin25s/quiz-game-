# 🎯 Quiz Game Using Shell Script

A simple command-line Quiz Game developed using **Bash Shell Script**.  
This project is designed for beginners to understand basic Linux shell scripting concepts like user input, conditional statements, and loops.

---

## 📌 Features

- Interactive command-line quiz
- Multiple-choice questions
- Instant feedback (Correct/Wrong)
- Final score display
- Lightweight and easy to run on any Linux system

---

## 🛠️ Technologies Used

- Bash Shell Script
- Linux Terminal
- Basic text editor (Nano / Vim / VS Code)

## 🚀 How to Run the Project

### 1. Clone the repository
```bash
git clone https://github.com/your-username/quiz-game-shell.git
2. Move into the directory
cd quiz-game-shell
3. Give execute permission
chmod +x quiz.sh
4. Run the script
./quiz.sh
🧠 How It Works
The script starts and displays a welcome message

Questions are shown one by one with options

User enters their answer

Script checks correctness using if condition

Score is updated accordingly

Final score is displayed at the end
